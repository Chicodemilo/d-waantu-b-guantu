# Agent naming convention

Agent names are human names (Archie, Mona, Pixel, Devin, Bolt, Sage). For projects using Claude Code teams, the agent role field matches the exact Claude teammate name (e.g. tester, frontend-worker, system-ops). Display format: Name/role/project (e.g. Sage/tester/DWB). Mapping: Archie=team-lead, Mona=pm, Pixel=frontend-worker, Devin=backend-worker, Bolt=system-ops, Sage=tester.
