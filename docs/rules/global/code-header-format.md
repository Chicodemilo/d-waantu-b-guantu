# Code header format

All Python source files must include a standard header block as the first comment after imports:

# Path: relative/path/to/file.py
# File: filename.py
# Created: YYYY-MM-DD
# Purpose: One sentence description
# Caller: What calls this
# Callees: What this calls
# Data In: Input params/types
# Data Out: Return types
# Last Modified: YYYY-MM-DD

Keep headers accurate when modifying files. Update Last Modified on every change.
